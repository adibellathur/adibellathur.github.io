News Classifier

Machine Learning and NLP based classification of News Articles

A simple Classifier to take the words in an article, and use the nouns to group articles by topic. I wrote scripts to semi-autonomously pull news titles to generate training sets for the naïve-bayes classifier. These titles are then fed into the classifier, which later parses and cleans words in title to figure out the broad topic of the article

  • Libraries Used:
  • TextBlob - Basic NLP
  • Requests - API Pulls

Banana Tracker

Object Tracking with OpenCV

Uses OpenCV to recognize color and shape, then follows object around on screen. Uses basic filtration techniques such as HSV filtering and noise reduction to improve reliability in varied light environments. Works best on bananas.

  • Libraries Used:
  • OpenCV3 - Vision processing

Claw Machine

A Classic Arcade Game, Augmented with Motion Controls

Built a fully functional Claw Machine using an Arduino, stepper motors, timing belt, wood, and custom 3D printed parts. The claw is controlled with a Myo, an armband that reads electrical impulses from to determine what your hand is doing

  • Tools Used:
  • Arduino
  • Processing
  • Myo SDK
  • Solidworks

Make Some Noise

Sound Visualization

A program to convert sounds into a visual art piece, mimicing the rippling of drops on water. Using input from a microphone or directly from a midi instrument (or anything with a headphone jack) the program sses a fourrier transform on the audio data, and uses frequency and volumes to determine where ripples will show and what colors they will be

  • Tools Used:
  • Processing
